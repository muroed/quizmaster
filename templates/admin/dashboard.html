Admin Instructions
Creating a Quiz:
  1. Click the "Create New Quiz" button above.
  2. Enter a title and optional description for your quiz.
  3. Add questions with various types: text, single-choice, or multiple-choice.
  4. For each question, you can add images, audio, or video content.
  5. Set the correct answers and point values for each question.
Managing Quizzes:
  1. Edit existing quizzes to update questions, answers, or media.
  2. Delete quizzes that are no longer needed.
  3. View the list of all your quizzes in the table above.
