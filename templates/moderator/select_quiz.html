How to Run a Quiz
  1. Start a new game by selecting one of the available quizzes.
  2. After starting a game, you'll get a unique game code that participants can use to join.
  3. Wait for participants to join your game using the code.
  4. When everyone is ready, you can start the quiz and control the flow of questions.
  5. Award extra points to teams when necessary, especially for text answers that need manual verification.
  6. View the leaderboard at any time to check team rankings.
  7. End the game when all questions have been answered or when you decide to conclude the quiz.
